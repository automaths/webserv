WEBSERV CONFIGURATION FILE GUIDELINES:

Configuration rules: 

    - listen + value (address) + ":" + value (port)
        exemple: "listen 127.0.0.1:8000"

    - client_body_buffer_size + value
        exemple: "client_body_buffer_size 200M;"
    
    - error_page + numero (+ numero + ...) + path
        exemple: "error_page 404 403 402 /one.html;"

    - root + path
        exemple: "root /my_root;"
    
    - index + path (+ path)
        exemple: "index index.html index.htm index.php;"
    
    - autoindex + value (on or off)
        exemple: "autoindex on;"
    
    - cgi + pair extension-path (+ pair + ...)
        exemple: "cgi .php /usr/lib/cgi-bin/php .py /usr/lib/cgi-bin/python;"

    - server_name + name (+ name + ...)
        exemple: "server_name coucou hola halo;"

    - allow_method + method (+ method + ...)
        exemple: "allow_method GET POST;"
    

Http scope rules: 

    - client_body_buffer_size
    - error_page
    - root
    - index
    - autoindex
    - cgi

Server scope rules:

    - listen
    - server_name
    - index
    - client_body_buffer_size
    - autoindex
    - root
    - allow_method
    - cgi
    - error_page

Location scope rules:

    - client_body_buffer_size
    - error_page
    - root
    - index
    - autoindex
    - allow_method
    - cgi
